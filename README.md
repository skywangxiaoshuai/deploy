### 注意

    项目正式运行前的准备：写进seed，rails db:seed
    1、添加平台管理员角色，并为该角色赋予权限
    2、在platforms表添加平台信息，并为平台创建平台管理员帐号。
    3、为平台管理员帐号添加平台管理员角色。
    
    添加代理商之前，roles表中需要有 代理商管理员 角色，且已经为该角色配置了对应的权限
    添加商户之前， roles表中需要有 商户管理员 角色， 且已经为该角色配置了对应的权限
